Jam Carousel Plugin

How to use

  1. 1. Add to your project jam-carousel.css file and link it in the head part of your main html.
  2. 2. Add to your project jam-carousel.js file and link it after the body part of your main html.
  3. 3. Gather all needed elements to make Jam of inside some wrapper.
  4. 4. Give this wrapper data attribute, named data-jam.
  5. 5. Set to the data-jam attribute any value you want.
  6. 6. Now add this part of script to your js:
    let jam1 = new Jam ({ data: "jam1" });
    The "jam1" is the value, you have set for data-jam attribute in part 5.
  7. 7. Make sure you have added the script from part 6 right after linking jam-carousel.js.
  8. 8. You can also use some additional styling of your Jam - all the properties are listed nearby. Have fun!
